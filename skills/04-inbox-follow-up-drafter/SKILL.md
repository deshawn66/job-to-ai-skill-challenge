# Inbox Follow-Up Drafter

Draft follow-up messages without sending them.

## Inputs

- Original message or thread summary
- Desired outcome
- Relationship context
- Tone preference
- Any deadline or next step

## Workflow

1. Identify what the other person needs.
2. Identify what the user wants to accomplish.
3. Draft the clearest useful response.
4. If the topic is sensitive, include a softer version.
5. Keep the message human, specific, and action-oriented.

## Output

Return:

- Recommended draft
- Optional softer draft
- Subject line, if email
- One-line reason the draft works

## Safety

- Draft only. Never send.
- Do not promise outcomes the user did not approve.
- Do not include confidential details in public examples.
- If legal, financial, medical, or regulated claims appear, flag for human review.
