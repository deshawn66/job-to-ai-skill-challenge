# Skill 04: Inbox Follow-Up Drafter

## What It Does

Drafts thoughtful follow-up messages from email or chat context without sending anything.

## Who It Helps

Busy operators who need faster, clearer follow-up.

## Inputs Needed

- Original message
- Desired outcome
- Relationship context
- Tone preference
- Any deadline or next step

## Tools It May Interact With

- Email
- GoHighLevel
- Slack
- Google Chat
- Notes

## Prompt

Draft a follow-up message based on this thread. Keep it clear, warm, and action-oriented. Do not send anything. Do not promise outcomes I did not approve. Include a short subject line if this is an email. Give me one recommended draft and one softer version if the topic is sensitive.

## Sample Input

A partner asked for an update on an open item. I need to acknowledge the delay, confirm we are working on it, and ask one clarifying question.

## Sample Output

Subject: Quick update on the open item

Hi [Name],

Thanks for checking in. We are still working through it and I wanted to make sure we do not give you a half-answer.

One quick question: do you need the final answer today, or would tomorrow morning still work?

Thanks,
[Your Name]

## Safety Notes

- Draft only. Never send.
- Do not make commitments the user did not approve.
- Do not include private customer details in shared examples.
