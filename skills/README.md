# Skill Index

These are example skills based on the kind of work an operator, founder, sales leader, or mortgage executive might do.

Use them as models. Replace the examples with your own job.

## The 12 Example Skills

1. `01-daily-work-capture` - Turns a messy workday into a clean task log.
2. `02-repeat-task-finder` - Finds repeatable work that is worth automating.
3. `03-meeting-to-action-plan` - Converts meeting notes into owners, decisions, and next steps.
4. `04-inbox-follow-up-drafter` - Drafts follow-ups without sending anything.
5. `05-tool-workflow-brief` - Explains how one business tool talks to another.
6. `06-scorecard-explainer` - Turns a report or dashboard into plain English.
7. `07-coaching-prep` - Prepares a coaching conversation.
8. `08-playbook-builder` - Turns a repeatable business motion into a playbook.
9. `09-compliance-first-pass` - Flags risky claims before public content ships.
10. `10-founder-research-and-feedback` - Reviews AI startups and drafts founder feedback.
11. `11-linkedin-build-in-public-post` - Turns the work into a public post.
12. `12-scary-tool-review` - Forces an honest review of an intimidating tool.

## How To Write Your Own

Each skill should answer:

- What job pain does this solve?
- Who is it for?
- What inputs does it need?
- What tools does it interact with?
- What does good output look like?
- What should it never do?
