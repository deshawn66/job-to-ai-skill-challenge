## Intro Section

Most AI advice starts with tools.

This project starts with the job.

I wrote down how I actually spent my day: meetings, follow-ups, reports, decisions, research, coaching, public content, and tool testing. Then I asked one question:

> Which parts of this work could become reusable skills?

The answer became 12 agents.

Each one has a job, an input, an output, a tool boundary, and a safety rule.

## Agent Card Fields

Each card should include:

- Agent name
- Job pain solved
- What it helps with
- Tools it may touch
- What the user gives it
- What it produces
- Safety boundary
- GitHub link

## Agent Cards

1. Daily Work Capture - turns a messy day into a clean task log.
2. Repeat Task Finder - finds which tasks are worth automating.
3. Meeting To Action Plan - turns notes into decisions and next steps.
4. Inbox Follow-Up Drafter - drafts follow-ups without sending anything.
5. Tool Workflow Brief - explains business systems in plain English.
6. Scorecard Explainer - turns reports into useful business insight.
7. Coaching Prep - prepares focused coaching conversations.
8. Playbook Builder - turns repeatable work into a usable playbook.
9. Compliance First Pass - flags risky claims before publishing.
10. Founder Research And Feedback - reviews AI startups from a real job perspective.
11. LinkedIn Build-In-Public Post - turns real work into public learning.
12. Scary Tool Review - reviews intimidating tools honestly.
